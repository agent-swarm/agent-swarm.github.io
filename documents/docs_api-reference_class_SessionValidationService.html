SessionValidationService

Service for managing and validating sessions within the swarm system. Tracks session associations with swarms, modes, agents, histories, storages, and states, ensuring session existence and resource usage consistency. Integrates with SessionConnectionService (session management), ClientSession (session lifecycle), ClientAgent (agent usage), ClientStorage (storage usage), ClientState (state usage), SwarmSchemaService (swarm association), and LoggerService (logging). Uses dependency injection for the logger and memoization for efficient validation checks.
