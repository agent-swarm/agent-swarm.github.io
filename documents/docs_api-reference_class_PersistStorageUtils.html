PersistStorageUtils

Implements IPersistStorageControl

Utility class for managing storage persistence per client (SessionId) and storage name (StorageName) in the swarm system. Provides methods to get/set storage data with a customizable persistence adapter.

Memoized function to create or retrieve storage for a specific storage name. Ensures a single persistence instance per StorageName, optimizing resource use.

Retrieves the data for a client from a specific storage, falling back to a default if unset. Accesses persistent storage for a SessionId under a StorageName (e.g., user records).
