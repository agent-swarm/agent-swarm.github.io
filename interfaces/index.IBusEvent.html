Interface IBusEvent

Interface representing a structured event for the internal bus in the swarm system. Extends IBaseEvent with a specific schema, used extensively in ClientAgent’s bus.emit calls (e.g., "run", "commit-user-message") to notify the system of actions, outputs, or state changes. Dispatched via IBus.emit to broadcast detailed, agent-driven events with input/output data and context.

interface IBusEvent {
    clientId: string;
    context: Partial<IBusEventContext>;
    input: Record<string, any>;
    output: Record<string, any>;
    source: EventBusSource;
    type: string;
}
